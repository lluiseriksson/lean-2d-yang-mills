# Milestones

## lean-2d-yang-mills — sandbox exactamente soluble

- M0: **cerrado para el modelo concreto SU(2)**. Heat kernel
  `K_t(g)=Σ_n (n+1)χ_n(g)e^{-t n(n+2)/4}`, puente Haar--esfera, ley orbital a
  todos los órdenes, convolución traducida de caracteres y semigrupo infinito.
- M1: **cerrado localmente**. `su2Migdal_subdivision_invariant` integra una
  arista compartida por dos caras y sustituye exactamente `(s,t)` por `s+t`.
  La reducción combinatoria de una celulación planar arbitraria sigue abierta.
- M2: **cerrado para el lazo simple de una cara**.
  `su2_exact_simpleLoop_areaLaw` prueba el exponencial de Casimir para todo
  label y toda área positiva. No debe confundirse con la cota previa de
  acoplamiento fuerte del repositorio madre.
- M3: statements-first del límite continuo (Lévy; Sengupta; Driver 1989).
- M4: zeta de Witten ζ_G(s)=Σ_λ dim(λ)^{-s}: convergencia y su papel en la
  función de partición sobre superficies.
